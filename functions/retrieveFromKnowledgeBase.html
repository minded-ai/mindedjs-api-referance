      Retrieve relevant documents from a knowledge base without generation.

      This function queries a knowledge base and returns matching document chunks based on the given query, with support for filtering and controlling the number of results.

      Parameters

      • params: {
            environment: string;
            filters?: KnowledgeBaseFilter;
            knowledgeBaseId?: string;
            numberOfResults?: number;
            query: string;
        }

        The retrieval parameters

        • environment: string

          The environment (e.g., 'development', 'production')

        • Optionalfilters?: KnowledgeBaseFilter

          Optional filters for metadata-based filtering

        • OptionalknowledgeBaseId?: string

          Optional ID of the knowledge base to query. If omitted, the backend will auto-select the knowledge base if exactly one exists for the agent. If 0 or >1 knowledge bases exist, an error will be thrown requiring you to pass knowledgeBaseId explicitly.

        • OptionalnumberOfResults?: number

          Optional number of results to return (default: 5)

        • query: string

          The search query string

      • timeoutMs: number = 30000

        Optional timeout in milliseconds (default: 30000)

      Returns Promise<
          { nextToken?: string; retrievalResults: KnowledgeBaseRetrievalResult[] },
      >

      Promise that resolves with retrieval results including content, metadata, and scores

      Throws

      When the retrieval fails or when knowledgeBaseId is omitted and 0 or >1 KBs exist

      Example

      import { retrieveFromKnowledgeBase } from '@minded-ai/mindedjs';

      // With explicit knowledgeBaseId
      const results = await retrieveFromKnowledgeBase({
        knowledgeBaseId: 'kb-123',
        environment: 'production',
        query: 'How do I reset my password?',
        numberOfResults: 10,
        filters: {
          equals: { key: 'category', value: 'support' }
        }
      });

      // Without knowledgeBaseId (auto-selects if agent has exactly 1 KB)
      const results2 = await retrieveFromKnowledgeBase({
        environment: 'production',
        query: 'What are the shipping options?',
        numberOfResults: 5
      });

      for (const result of results.retrievalResults) {
        console.log(result.content.text, result.score);
      }
