      Register a handler for timer trigger events for a specific timer name. The handler will be called when the specified timer expires for any session.

      Note: Only one handler per timer name is supported. Calling this function multiple times with the same timer name will replace the previous handler.

      Example

      import { onTimer } from '@minded-ai/mindedjs';

      // Register a handler for the 'followup' timer
      onTimer({
        timerName: 'followup',
        handler: async ({ sessionId, payload }) => {
          console.log(`Followup timer triggered for session ${sessionId}`);

          // Handle the followup timer
          await agent.invoke({
            triggerName: 'timerFollowup',
            triggerBody: { message: payload.message },
            sessionId
          });
        }
      });
